## English

### Overview
This project is a web scraper designed to retrieve and process data from [EVE Reference](https://everef.net/). The scraper uses web scraping techniques to extract information from the EVE Online universe database.

### How to Use
1. Open the project files and locate the `start_request` function
2. Modify the `urls` list with the URLs you want to retrieve from https://everef.net/
3. Run the scraper to collect the data
4. The extracted data will be processed according to your configuration

### Features
- Web scraping from EVE Reference (everef.net)
- Configurable URL list for flexible data collection
- Data extraction and processing capabilities

### Requirements
- Python 3.x
- Required dependencies (specify in requirements.txt if available)
